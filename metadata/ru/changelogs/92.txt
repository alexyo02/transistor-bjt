# v4.1.2 - Modern Love

**03.01.2023**

⚠️ В этом обновлении служба фонового воспроизведения была полностью переработана. Транзистор теперь использует Media3. Я использовал пересборку, чтобы избавиться от нехарактерных для приложений такого типа фич: кнопки отмены (x) на шторке и запуска приложения в фоновом режиме ярлыком на Главном экране. Извините, если вам понравилась одна из этих функций. Для меня важно, чтобы Транзистор соответствовал требованиям платформы. Это значительно упрощает поддержку моего любимого радиоприложения.

- Транзистор теперь использует библиотеку воспроизведения media3;
- обновлены переводы;
- включена функция настройки языка для каждого приложения (Android 13+).
